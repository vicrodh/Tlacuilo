HIGHLIGHTS - WHERE TLACUILO WINS OR TIES
================================================

Multi-tab: Only Okular and LibreOffice = Tlacuilo supports 10+ tabs with LRU
Advanced annotations: All have them = Tlacuilo has stamps, shapes, arrows, typewriter comparable
Context menu text: Only MasterPDF, Nitro, Adobe = Tlacuilo has Copy, Highlight, Underline, Search
Bookmarks + Outlines: Okular = Tlacuilo has hierarchical outlines with precise scroll
Search in bookmarks/annotations: Only MasterPDF, Nitro = Tlacuilo has filter in bookmarks and annotations
Attachments viewer: Only MasterPDF = Tlacuilo is unique in viewer with attachments viewer
Document info viewer: Only Tlacuilo = Dedicated tab with complete metadata
Signatures placeholder: All have them = Tlacuilo has UI structure for future integration
Nord theme: Only Tlacuilo = Modern and consistent UX
Svelte 5 + Rust: Only Tlacuilo = Modern, fast stack


WHERE TLACUILO LAGS BEHIND
==================================

Text editing: MasterPDF, Nitro, LibreOffice = Tlacuilo: Core of editor (0%)
